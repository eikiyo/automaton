# Guidance: Data & Sample (D07)
## Gates: D07-01 through D07-10 | Critical: D07-01, D07-02

### What JIBS Demands
- D07-01 (CRITICAL): Data source described for independent replication (database, version, access date, filters).
- D07-02 (CRITICAL): Sample size reported and sufficient for statistical tests.
- D07-03: Sampling procedures and inclusion/exclusion criteria stated.
- D07-04: Time period stated and justified.
- D07-05: Country sample reflects theoretical population, not data convenience.
- D07-06: Selection biases identified and addressed.
- D07-07: Missing data reported (volume, pattern, handling method).
- D07-08: Descriptive statistics table (means, SDs, min, max, N).
- D07-09: Correlation matrix for all key variables.
- D07-10: Target population for generalization stated.

### How to Pass (from ARA Protocol & Writer phases)

**For Review/Conceptual Papers (what the agent writes):**
Describe the search methodology as data:
- Databases searched (Semantic Scholar, OpenAlex, CrossRef, PubMed, CORE).
- Search terms and Boolean operators.
- Date range and language restrictions.
- Inclusion/exclusion criteria table.

**Screening Methodology (HONEST REPORTING):**
State: "Title and abstract screening was conducted using automated relevance scoring, with papers scoring above the threshold selected for full-text review."
Acknowledge limitation: "Screening was AI-assisted rather than independent human dual-reviewer screening."

**PRISMA Flow (D07-01):**
Report the PRISMA pipeline:
- Records identified from databases.
- Duplicates removed.
- Records screened.
- Records excluded (with reasons).
- Full-text articles assessed.
- Studies included in synthesis.

**Study Characteristics Table (D07-08):**
| Author/Year | Country | Design | Sample Size | Population | Key Variables | Main Finding |

**Quality Assessment (D07-02):**
Use JBI Critical Appraisal framework. Report risk of bias per study.
