# Guidance: Hypotheses & Argumentation (D06)
## Gates: D06-01 through D06-08 | Critical: D06-01, D06-02, D06-03

### What JIBS Demands
- D06-01 (CRITICAL): All hypotheses falsifiable — could be empirically wrong.
- D06-02 (CRITICAL): Each hypothesis derived from theoretical framework, not intuition.
- D06-03 (CRITICAL): Logical chain premise → argument → hypothesis traceable without leaps.
- D06-04: Directional hypotheses justified.
- D06-05: All stated hypotheses tested — none abandoned.
- D06-06: Interaction/moderation theoretically derived, not post-hoc.
- D06-07: Mediation fully argued (X→M→Y), not just tested.
- D06-08: Null or competing predictions discussed where theory permits.

### How to Pass (from ARA Hypothesis & Critic phases)

**Hypothesis Structure:**
Format: "Proposition N: [clear, testable statement]"
Each must include:
1. Theoretical rationale (which theories support this).
2. Mechanism description (the causal chain).
3. Supporting evidence from literature.
4. Boundary conditions.
5. Illustrative example (real-world scenario).

**Critic's 6-Test Protocol (apply to EACH hypothesis):**
1. NOVELTY KILL TEST: Does any paper already answer this?
2. FEASIBILITY: Can this actually be studied?
3. SO-WHAT TEST: If confirmed, does it change practice or theory?
4. EVIDENCE QUALITY: What GRADE rating does supporting evidence have?
5. ADVERSARIAL COUNTER-HYPOTHESIS: What's the strongest objection?
6. PUBLICATION VENUE: Could this be published in Q1/Q2 journal?

**Calibration Rule:**
If you approve more than 40% of hypotheses, standards are too low. Target 2-4 surviving out of 10-15 candidates.

**For Conceptual Papers (Propositions, not Hypotheses):**
One subsection per proposition:
1. Proposition statement.
2. Theoretical rationale.
3. Mechanism description.
4. Supporting evidence.
5. Boundary conditions.
6. Illustrative example.
