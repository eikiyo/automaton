# Guidance: Results Presentation (D12)
## Gates: D12-01 through D12-07 | Critical: D12-01

### What JIBS Demands
- D12-01 (CRITICAL): Coefficient estimates, standard errors/CIs, significance levels all reported.
- D12-02: Effect sizes alongside p-values.
- D12-03: Economic/practical significance discussed.
- D12-04: Null results reported with explanation.
- D12-05: All model specifications presented.
- D12-06: Tables readable with clear headers and footnotes.
- D12-07: No p-hacking signal.

### How to Pass (from ARA Writer & Synthesis phases)

**For Review Papers — Organize by THEME:**
Each theme becomes a subsection containing:
1. Theme statement (one clear sentence).
2. Evidence summary with citations.
3. Effect sizes and sample sizes (where available).
4. Contradicting evidence (with explanation).
5. Evidence quality note (GRADE rating).
6. Summary table for the theme.

At least 70% of Results paragraphs MUST contain a specific number (effect size, CI, OR, N, or %).

**For Conceptual Papers — One subsection per proposition:**
1. Proposition statement: "Proposition N: [statement]"
2. Theoretical rationale.
3. Mechanism description.
4. Supporting evidence.
5. Boundary conditions.
6. Illustrative example.

**Tables (D12-06):**
Required tables:
- Table 1: Study Characteristics (Author/Year, Country, Design, Sample, Key Variables, Finding).
- Table 2: GRADE Evidence Summary.
- Table 3: Risk of Bias Assessment.
- Table 4+: Theme-specific evidence synthesis.

**Quantitative Summary:**
| Outcome | N Studies | Total N | Effect Size Range | Median Effect | Direction Consistency |
If no quantitative data: "No quantitative data — qualitative synthesis only."
