# Guidance: Statistical Methods (D10)
## Gates: D10-01 through D10-10 | Critical: D10-01, D10-02

### What JIBS Demands
- D10-01 (CRITICAL): Statistical method appropriate for data structure.
- D10-02 (CRITICAL): Endogeneity addressed or acknowledged with direction of bias.
- D10-03: Model assumptions stated and tested.
- D10-04: Standard errors corrected for heteroscedasticity/clustering.
- D10-05: Multicollinearity assessed (VIF < 10).
- D10-06: FE vs RE choice justified (Hausman test).
- D10-07: Mediation uses bootstrapped CIs, not Baron-Kenny alone.
- D10-08: Multilevel data uses HLM/mixed effects.
- D10-09: Estimation strategy in dedicated Methods section.
- D10-10: Software and version reported.

### How to Pass

**For Review/Conceptual Papers:**
Describe the synthesis methodology:
- Narrative synthesis, thematic analysis, or meta-analysis approach.
- GRADE framework for evidence certainty assessment.
- How heterogeneity is handled.
- Subgroup analyses planned.

**Write in PAST TENSE** — the analysis has already been done.

Report software used (e.g., "Systematic search conducted using Semantic Scholar and OpenAlex APIs").
